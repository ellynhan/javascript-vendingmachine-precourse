### 구현할 기능 목록

1. Utils폴더에 선택자와 상수 정리하기
2. **화면구성**: 상단에 자판기 제목, 상품관리, 잔돈 충전, 상품구매 탭 메뉴를 추가한다.
3. **화면구성**: 금액 투입, 구매할 수 있는 상품현항, 잔돈 화면을 구현한다.
4. **상품관리**: 상품명, 가격, 수량을 입력하여 상품을 추가할 수 있어야한다.
5. **상품관리**: 상품가격은 100원 부터 시작하고 10원으로 나누어 떨어져야한다.
6. **상품관리**: 사용자에게 추가한 상품이 보여져야한다.
7. **잔돈충전**: 사용자가 입력한 충전 금액이 기존 자판 보유금액과 더해져 보여져야한다.
8. **잔돈충전**: 자판기 보유 금액만큼 동전이 무작위로 생성되어야한다.
9. **상품구매**: 사용자로부터 투입 금액을 입력받는다.
10. **상품구매**: 사용자로부터 입력받은 투입 금액은 10원으로 나누어 떨어져야한다.
11. **잔돈계산**: 잔돈을 돌려줄 때 보유한 최소 개수의 동전으로 잔돈을 돌려준다.
12. **잔돈계산**: 잔돈을 반활할 수 없는 경우 반환 가능한 금액만 반환한다.
13. **공통**: 다른 탭으로 이동했다 돌아와도 기존 탭의 상태가 최근 상태로 유지되어야한다
